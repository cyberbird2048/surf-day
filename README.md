# 今天冲浪吗 🏄 — 自用冲浪浪报小程序

单文件 H5（`index.html`），无依赖、无构建、无 API key。给自己和朋友用。

## 功能

- **浪报**：18 个浪点（国内 11：日月湾/后海湾/西冲/红海湾/石老人/富力湾/汕头黄金海岸/山海里双月湾/六鳌/舟山东沙/青岛两河口；全球 7 个默认折叠：巴厘岛×2/锡亚高/斯里兰卡/千叶/黄金海岸/夏威夷），每点一张卡：当前浪高/涌浪周期/风（离岸·向岸·侧风）+ 0–100 评分 + **今日最佳下水窗口**
- **详情页**：未来 48 小时逐时评分条、今/明/后三天最佳窗口（含峰值时刻）、今日潮汐曲线、海温→湿衣建议、7 日概览、一键分享浪报文本、地图导航、实况摄像头 & 权威浪报外链（Surfline/Surf-Forecast/Swellnet/BCM/Balicams 等，已逐一验证）
- **选板**：短板路线（容积法，系数对标 Firewire/Boardcave）+ 长板路线（长度为主），身高/体重/水平/体能 → 推荐板容积或长度 + 板型区间（软板/长板/Funboard/鱼板/短板）
- **其他**：新手/进阶/高手三档评分标准、收藏置顶、按评分自动排序（可切回固定顺序）、30 分钟本地缓存、离线快照兜底、深色模式、设计语言：同一条曲线 VI v1.0
- **首启水平引导**：首次打开弹出一次性档位选择，联动评分、排序与选板的默认推荐

## 评分逻辑

`得分 = 浪高匹配度×45% + 涌浪周期×25% + 风×30%`（按水平调整浪高理想区间；离岸风加分、向岸风重罚）。
最佳窗口 = 白天里连续 ≥55 分的时段，收窄到峰值 −8 分以内。≥72 冲！/ ≥55 不错 / ≥38 凑合 / <38 算了。

## 数据

[Open-Meteo](https://open-meteo.com) Marine + Forecast API，免费免注册，浏览器直接跨域拉取。
所有时间均为**浪点当地时间**。内置一份 72h 数据快照，断网/被墙时自动降级并显示提示。
外链资源经脚本+人工双重验证，有摄像头的浪点（乌鲁瓦图/苍古/韦利格默/一宫/Snapper）卡片上有 📷 标识。
国内浪点直播指引：微信小程序「爱冲浪Isurf」（无网页版，故不做外链）。

## 怎么用 / 给朋友

1. **在线打开**：直接访问上面的链接，手机浏览器里「添加到主屏幕」就是一个小程序
2. **本地开**：双击 `index.html`（Safari/Chrome 均可），AirDrop 或微信发文件给朋友即可
3. **自己部署**：仓库根目录就是站点内容，任意静态托管（GitHub Pages/Cloudflare Pages/Vercel）直接指向仓库根即可

## 维护

- 加浪点：改 `index.html` 里的 `SPOTS` 数组（lat/lon/facing 朝海方向角度），并重跑快照注入
- 刷新内置快照：`node inject-snapshot.mjs index.html`（加浪点后记得同步改脚本里的坐标表）
- push 到 main 分支后 GitHub Pages 会自动重新发布（约 1 分钟生效）
